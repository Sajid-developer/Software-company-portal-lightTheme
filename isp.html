How ISP Service Works?

The M-Cash system's functional architecture is structured around several key elements:

  • System Actors: This includes all entities interacting with M-Cash, from individual users like subscribers and agents to third-party applications such as the Amal Express remittance platform.
  • Access Channels: Detailed descriptions of how users connect to the system via USSD, SIM App, and Web interfaces.
  • Wallets and Accounts: Information on the types of virtual wallets and accounts available within the system.
  • Registration Process: Steps required for users to set up their mWallet and access services.
